# MonadArena 

Contract: 0xe65A8f1C8100a0437cA106685Cb240C43d721F7d

A Web3 platform on the Monad blockchain where users can discover live hackathons, watch project pitch videos, and stake tokens to predict winners. After results are announced, rewards are automatically split—80% to correct predictors and 20% to the winning team—creating an engaging, incentive-driven ecosystem for both builders and supporters.
